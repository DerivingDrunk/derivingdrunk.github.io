Multiple Intelligences Programs

Musical

We would create a band program, where kids can learn to play different instruments. At the end of the year, we will have a talent show where they can perform.

  • This program will allow the kids to express themselves with music. They will learn to play songs that they like.

    • Body-Kinesthetic

    We would create a dance program. Kids will learn different dances to songs from musicals. We would include them a talent show at the end of the year.

  • This program will allow kids to get exercise throughout the day. They will also be improving their hand-eye coordination.

    • Linguistic

    We would create a writing program for older kids. They will write poems and short stories. These would be featured in a book that is given to everyone at the end of the year.

  • This program will allow kids to express their creativity. They can use their imagination to create different kinds of stories.

    • Logical-Mathematical

    We would create a math and logic program for older kids. This would operate both as a mini math camp and a program for logical thinking. On the math side, they would learn some slightly more advanced information than their current knowledge. On the logic side, we will go feature puzzles and other exercises that force them to think logically.

  • This program will help foster the growth of a child’s brain by allowing them to challenge themselves with more complex math problems and logic problems.

    • These elements relate to Howard Gardner's Theory of Multiple Intelligences.
